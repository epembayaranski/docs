Data fakultas berasal dari hasil integrasi eRegistrasi, data fakultas digunakan sebagai data fakultas dalam aktivitas di aplikasi ePembayaran. Pengguna ePembayaran hanya dapat melakukan view, tanpa mengubah data fakultas, pengubahan data fakultas dapat dilakukan melalui pengubahan di aplikasi eRegistrasi
