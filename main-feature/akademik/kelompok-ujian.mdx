---
title: "Kelompok Ujian"
description: "Referensi Kelompok Ujian untuk mengelola data pembayaran"
---

Admin dapat melihat data kelompok ujian di aplikasi ePembayaran. Data Kelompok Ujian yang disediakan adalah hasil integrasi dengan aplikasi eRegistrasi, dimana Kelompok Ujian yang terdapat pada aplikasi eRegistrasi akan muncul juga di dalam aplikasi ePembayaran. Referensi kelompok ujian digunakan untuk pengelolaan data pembayaran mahasiswa.

<AccordionGroup>
  <Accordion title="Data Kelompok Ujian" icon="layer-group">
    Data referensi kelompok ujian terdirii dari kode dan nama kelompok ujian
  </Accordion>
  <Accordion title="Edit Nama dan Kode" icon="pencil">
    Pengguna dapat mengubah nama dan kode kelompok ujian. Cara untuk mengubah nama dan kode yaitu klik tombol edit pada daftar kelompok ujian, akan muncul pop up form untuk mengubah data, isikan data nama dan kode yang baru, klik simpan.
  </Accordion>
</AccordionGroup>
