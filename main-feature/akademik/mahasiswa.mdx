Pengguna dapat melihat daftar referensi mahasiswa dari hasil integrasi dengan eRegistrasi dan eAkademik. Data yang tampil adalah data calon mahasiswa dan mahasiswa.

<AccordionGroup>
  <Accordion title="Data Mahasiswa" icon="book-user">
    Data disajikan dalam tabel, lengkap dengan informasi seperti

    - NIM
    - No Test
    - No Pendaftaran
    - Nama Mahasiswa, Informasi Status Mahasiswa dan Angkatan
    - Program Studi, Jenjang dan informasi fakultas
    - Gelombang masuk
    - Waktu kuliah
    - UKT
  </Accordion>
  <Accordion title="Export data" icon="file-excel">
    Pengguna dapat melakukan ekspor data mahasiswa menjadi file Excel. Pengguna juga dapat memilih data dengan filter yang disediakan.
  </Accordion>
  <Accordion title="Searchbar" icon="search">
    Pengguna dapat mencari data mahasiswa dengan input keyword ke searchbar yang ada di atas tabel data mahasiswa. Pengguna dapat memasukkan keyword sesuai NIM, No Test dan Nama mahasiswa
  </Accordion>
</AccordionGroup>
