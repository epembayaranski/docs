Data program studi digunakan untuk aktivitas transaksi di ePembayaran, data ini diambil dari hasil integrasi dengan aplikasi eRegistrasi.

Pada halaman referensi akademik program studi, pengguna dapat melakukan pencarian menggunakan searchbar (input keyword), atau menggunakan filter yang disediakan. Pengguna juga dapat mengubah data program studi (hanya kode program studi) dengan cara klik tombol edit

Pengguna tidak dapat mengubah nama program studi. Pengubahaan nama program studi dapat dilakukan di aplikasi eRegistrasi.
