Pengguna dapat melihat refrensi data UKT pada halaman UKT di menu Akademik, sub menu UKT. Daftar UKT diambil dari data di aplikasi eRegistrasi, Referensi data UKT ditarik dari hasil integrasi di eRegistrasi. Data UKT digunakan dalam aktivitas pembayaran untuk mengelompokkan data mahasiswa berdasarkan UKT dan lainnya.
