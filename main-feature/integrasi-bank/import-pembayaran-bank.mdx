Menu ini digunakan untuk melakukan impor data pembayaran dari bank ke sistem ePembayaran, dengan fitur ini pengguna dapat mengunggah file dari sistem bank, kemudian disesuaikan dengan format import yang tersedia di ePembayaran, lalu pengguna dapat menyimpan data pembayaran dari bank tersebut untuk kemudian dilakukan validasi di menu validasi pemabayran bank.

### Berikut langkah-langkah untuk melakukan import data pembayaran

<Steps>
  <Step title="Masuk ke ePembayaran, pilih menu integrasi bank, pilih Import Pembayaran Bank">
    Akan tampil halaman import pembayaran bank sebagai berikut

    ![Halamanimportpembayaranbank 1](/images/Halamanimportpembayaranbank-1.png)

    Gambar. Halaman import pembayaran bank
  </Step>
  <Step title="Pilih field yang wajib diisi">
    Field dapat dipilih yaitu:

    **Periode :** Pilih periode sesuai periode pembayaran yang akan di import

    **Bank :** Pilih bank yang sesuai dengan data yang akan di import

    **Nomor bukti:** Isi field nomor bukti impor pembayaran bank

    **Keterangan :** Isi keterangan dengan pesan free text

    **Separator :** Pilih separator data pada file yang sesuai

    **File CSV :** Klik untuk upload file sesuai dengan format yang tersedia

    ![Excelimportpembayaranbank Template](/images/excelimportpembayaranbank_template.png)

    Gambar. Contoh format file impor pembayaran bank
  </Step>
