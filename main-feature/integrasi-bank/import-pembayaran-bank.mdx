Menu ini digunakan untuk melakukan import data pembayaran dari bank ke sistem ePembayaran, dengan fitur ini memungkinkan pengguna mengunggah file sesuai format yang disediakan, berisi data pembayaran tagihan dari bank.

![Halamanimportpembayaranbank](/images/Halamanimportpembayaranbank.png)

Gambar. Halaman import pembayaran bank

### Berikut langkah-langkah untuk melakukan import

<Steps>
  <Step title="Masuk ke ePembayaran, pilih menu integrasi bank, pilih Import Pembayaran Bank">
    Akan tampil halaman import pembayaran bank sebagai berikut
  </Step>
  <Step title="Pilih field yang wajib diisi">
    Field dapat dipilih yaitu:

    **Periode :** Pilih periode sesuai periode pembayaran yang akan di import

    **Bank :** Pilih bank yang sesuai dengan data yang akan di import

    Nomor bukti : Isi field nomor bukti

    **Keterangan : **Isi keterangan dengan pesan free text

    **Separator : **Pilih separator data pada file yang sesuai

    **File CSV :** Klik untuk upload file sesuai dengan format yang tersedia 
  </Step>
  <Step title="Pastikan data telah sesuai, klik simpan untuk upload file, pesan sukses akan muncul apabila aksi berhasil dilakukan.">
    
  </Step>
</Steps>
