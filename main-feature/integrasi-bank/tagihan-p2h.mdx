Menu ini digunakan pengguna melakukan rekap data tagihan yang telah dibuat di ePembayaran untuk kemudian di-\_input \_ke dalam sistem cms bank yang memiliki kerjasama dengan skema point-to-host. Pengguna dapat memlih data mana yang akan direkap kemudian unduh dengan format file excel.

### Langkah langkah untuk melakukan rekap data tagihan P2H

<Steps>
  <Step title="Masuk ke ePembayaran, Pilih menu integrasi bank, pilih Tagihan P2h">
    ![Halaman Tagihan P2h(empty)](/images/HalamanTagihanP2H\(empty\).png)

    Gambar.  Halaman Tagihan P2H
  </Step>
  <Step title="Seteleah halaman Tagihan P2H muncul, pilih filter data untuk menampilkan data sesuai dengan kebutuhan">
    Filter yang tersedia adalah sebagai berikut:

    ![Filtertagihanp2h](/images/filtertagihanp2h.png)

    Gambar. Filter menu Tagihan P2H
  </Step>
  <Step title="Jika terdapat data tagihan maka akan muncul informasi detail tagihan dan jumlah tagihan, klik tombol export untuk melakukan export tagihan P2H">
    ![Datatagihanditemukantagihanp2h](/images/Datatagihanditemukantagihanp2h.png)

    Gambar. Tagihan ditemukan, data siap di-_export_
  </Step>
</Steps>
