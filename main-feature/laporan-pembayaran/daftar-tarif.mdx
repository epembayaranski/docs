Menu laporan daftar tarif digunakan untuk melihat rekap daftar tarif yang dibuat oleh pengguna di aplikasi ePembayaran. Pengguna dapat melakukan rekap daftar tarif sesuai dengan periode, jenjang, fakultas, program studi, angkatan dan gelombang. Pengguna juga dapat melakukan export excel file (.xls) pada menu ini.

Langkah untuk melakukan export data daftar tarif pembayaran adalah sebagai berikut

    - Periode, 
    - Jenjang, 
    - Fakultas, 
    - Program studi, 
    - Angkatan,
    - Gelombang.
