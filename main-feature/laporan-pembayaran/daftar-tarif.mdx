Menu laporan daftar tarif digunakan untuk melihat rekap daftar tarif yang dibuat oleh pengguna di aplikasi ePembayaran. Pengguna dapat melakukan rekap daftar tarif sesuai dengan periode, jenjang, fakultas, program studi, angkatan dan gelombang. Pengguna juga dapat melakukan ekspor file Excel (.xls) pada menu ini.

Langkah untuk melakukan export data daftar tarif pembayaran adalah sebagai berikut

<Steps>
  <Step title="Masuk ke aplikasi ePembayaran">
    
  </Step>
  <Step title="Pilih menu Laporan Pembayaran, pilih Daftar Tarif">
    ![Halamandaftartariflaporanpembayaran](/images/halamandaftartariflaporanpembayaran.png)

    Gambar. Halaman laporan daftar tarif
  </Step>
  <Step title="Pilih filter sesuai data yang dibutuhkan">
    Filter yang dapat dipilih adalah:

    - Periode,
    - Jenjang,
    - Fakultas,
    - Program studi,
    - Angkatan,
    - Gelombang

    ![Filterdaftartariflaporanpembayaran](/images/filterdaftartariflaporanpembayaran.png)

    Gambar. Filter laporan daftar tarif
  </Step>
  <Step title="Klik tombol export excel (.xls) pada bagian kanan atas tabel">
    File Excel berisi rangkuman daftar tarif akan otomatis terunduh
  </Step>
</Steps>
