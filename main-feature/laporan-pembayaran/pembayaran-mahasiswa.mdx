Menu laporan pembayaran mahasiswa digunakan untuk merekap seluruh aktivitas pembayaran yang di buat di ePembayaran.

Langkah-langkah untuk mendapatkan laporan pembayaran mahasiswa sesuai data yang dibutuhkan adalah sebagai berikut.

<Steps>
  <Step title="Masuk ke aplikasi ePembayaran">
    
  </Step>
  <Step title="Pilih menu laporan pembayaran, pilih pembayaran mahasiswa">
    
  </Step>
  <Step title="Pilih data yang dibutuhkan melalui fitur filter">
    Anda dapat menyeleksi data yang tampil dengan fitur filter data, adapun filter data yang disediakan yaitu

    - Periode Tahun
    - Tanggal awal dan akhir pembayaran
    - Angkatan
    - Jenjang
    - Fakultas
    - Program Studi
    - Jalur Masuk
    - Gelombang
    - Jenis Pembayaran
    - Jenis Biaya
    - Bank
    - Status Lunas

    ![Filterlaporanpembayaranmahasiswa Pn](/images/filterlaporanpembayaranmahasiswa.png)

    Gambar. Filter Laporan Pembayaran Mahasiswa
  </Step>
  <Step title="Data akan tampil sesuai filter yang dipilih, klik tombol export .xls untuk unduh file excel rekap laporan pembayaran">
    Anda dapat melakukan export file excel dari data yang dipilih dan export file pdf dengan klik tombol export (.xls) atau cetak (.pdf)

    ![Laporanpembayaranmahasiswapage Pn](/images/laporanpembayaranmahasiswapage.png)

    Gambar. Halaman Laporan Pembayaran Mahasiswa
  </Step>
</Steps>
