Laporan pendaftaran mencakup rangkuman seluruh aktivitas pembayaran pendaftaran mahasiswa pada proses penerimaan mahasiswa baru (PMB). Pembayaran pendaftaran termasuk pembelian token pendaftaran.

Berikut langkah langkah yang dapat dilakukan untuk melihat laporan pendaftaran sesuai dengan kebutuhan.

<Steps>
  <Step title="Masuk ke ePembayaran, pilih menu laporan pembayaran, pilih pendaftaran">
    
  </Step>
  <Step title="Pilih filter sesuai kebutuhan, klik tampilkan">
    Fiter yang tesedia yaitu:

    - Tanggal awal pembayaran
    - Tanggal Akhir Pembayaran
    - Loket
    - Status Pendaftaran
    - Periode
    - Gelombang
    - Jalur Masuk
    - Kelompok Ujian

    ![Filterlaporanpembayaranpendaftaran](/images/filterlaporanpembayaranpendaftaran.png)

    Gambar. Filter laporan pembayaran pendaftaran
  </Step>
  <Step title="Data laporan pembayaran akan muncul sesuai dengan filter data dipilih.">
    ![Halamanlaporanpembayaranpendaftara](/images/halamanlaporanpembayaranpendaftara.png)

    Gambar. Halaman laporan pembayaran pendaftaran

    <Tip>
      Pengguna juga dapat melakukan export data laporan pendaftaran dalam bentuk file excel (.xls) klik tombol export .xls, file excel akan otomatis terunduh.
    </Tip>
  </Step>
</Steps>
