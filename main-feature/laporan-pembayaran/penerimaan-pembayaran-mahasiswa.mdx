Menu penerimaan pembayaran mahasiswa digunakan admin untuk mendapatkan informasi detail mengenai seluruh pembayaran mahasiswa dari jenis pembayaran yang dibuat di ePembayaran lengkap dengan detail di setiap data mahasiswa.

Terdapat fitur list daftar penerimaan pembayaran permahasiswa, dari periode tertentu, rekap per fakultas, dan fitur export file excel

Untuk mendapatkan data penerimaan pembayaran mahasiswa, admin dapat melakukan langkah-langkah berikut:

<Steps>
  <Step title="Masuk ke aplikasi ePembayaran">
    
  </Step>
  <Step title="Pilih menu laporan pembayaran, pilih penerimaan pembayaran mahasiswa">
    Gambar. Halaman Laporan Penerimaan Pembayaran Mahasiswa
  </Step>
  <Step title="Pada fitur filter data, pilih data yang akan ditampilkan sesuai dengan fitur filter yang disediakan">
    
  </Step>
  <Step title="Admin akan mendapatkan data sesuai filter data, admin dapat melakukan export excel dengan klik tombol export .xls">
    Export excel akan menyesuaikan data yang ditampilkan, yang dipilih menggunakan fitur filter data.
  </Step>
</Steps>
