Menu penerimaan pembayaran mahasiswa digunakan admin untuk mendapatkan informasi detail mengenai seluruh pembayaran mahasiswa dari jenis pembayaran yang dibuat di ePembayaran lengkap dengan detail di setiap data mahasiswa.

Terdapat fitur list daftar penerimaan pembayaran permahasiswa, dari periode tertentu, rekap per fakultas, dan fitur export file excel

Untuk mendapatkan data penerimaan pembayaran mahasiswa, admin dapat melakukan langkah-langkah berikut:

<Steps>
  <Step title="Pilih menu laporan pembayaran, pilih penerimaan pembayaran mahasiswa">
    ![Halaman Penerimaan Pembayaran Mahasiswa](/images/HalamanPenerimaanPembayaranMahasiswa.png)

    Gambar. Halaman Laporan Penerimaan Pembayaran Mahasiswa
  </Step>
  <Step title="Pada fitur filter data, pilih data yang akan ditampilkan sesuai dengan fitur filter yang disediakan">
    ![Filter Penerimaan Pembayaran Mahasiswa](/images/FilterPenerimaanPembayaranMahasiswa.png)

    Gambar. Filter Laporan Penerimaan Pembayaran Mahasiswa

    Filter yang disediakan meliputi:

    - Jenjang : Jenjang pendidikan perguruan tinggi
    - Fakultas : Opsi Fakultas yang ada di perguruan tinggi
    - Program studi : Opsi tersedia sesuai dengan referensi yang ada di perguruan tinggi
    - Periode tahun: untuk memilih periode semester yang akan ditampilkan datanya
  </Step>
  <Step title="Admin akan mendapatkan data sesuai filter data, admin dapat melakukan export excel dengan klik tombol export .xls">
    ![Halaman Penerimaan Pembayaran Mahasiswa 2](/images/HalamanPenerimaanPembayaranMahasiswa-2.png)

    Gambar. Hasil Tampilan Data Sesuai Filter Dipilih

    ![Tabelfakultasdipenerimaanpembayaranmahasiswa](/images/tabelfakultasdipenerimaanpembayaranmahasiswa.png)

    Gambar. Rangkuman Penerimaan Pembayaran Mahasiswa Per Fakultas

    <Note>
      Pengguna dapat melihat data penerimaan pembayaran mahasiswa per fakultas pada tabel total penerimaan per fakultas, data disajikan yaitu nama fakultas dan total penerimaannya, serta total penerimaan seluruh fakultas.
    </Note>
    Export excel akan menyesuaikan data yang ditampilkan, yang dipilih menggunakan fitur filter data.
  </Step>
</Steps>
