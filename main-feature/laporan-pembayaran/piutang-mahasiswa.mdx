Menu piutang mahasiswa digunakan untuk merekap informasi tagihan piutang mahasiswa dari setiap jenis pembayaran yang ada di ePembayaran.

Admin dapat mengikuti langkah langkah berikut untuk mendapatkan data laporan piutang pembayaran mahasiswa.

<Steps>
  <Step title="Masuk ke Aplikasi ePembayaran">
    
  </Step>
  <Step title="Pilih menu laporan pembayaran, Pilih piutang mahasiswa">
    ![Laporanpiutangmahasiswa Pn](/images/laporanpiutangmahasiswa.png)

    Gambar. Halaman Piutang Mahasiswa
  </Step>
  <Step title="Untuk mendapatkan data sesuai dengan kebutuhan, pengguna dapat memilih filter data sesuai kebutuhan">
    Adapun filter data yang disediakan untuk menampilkan data piutang pembayaran mahasiswa yaitu:

    - Periode
    - Angkatan
    - Jenis Pembayaran
    - Jenjang
    - Program Studi
    - Fakultas
    - Jalur masuk
    - Gelombang

    ![Filterlaporanpiutangmahasiswa Pn](/images/filterlaporanpiutangmahasiswa.png)

    Gambar. Filter Laporan Piutang Mahasiswa
  </Step>
  <Step title="Pengguna juga dapat melakukan export data ke dalam file excel dengan klik tombol export .xls atau cetak .pdf">
    Pastikan pengguna sudah memilih data yang sesuai untuk ditampilkan menggunakan filter data yang tersedia, klik export .xls untuk mengunduh data yang dipilih dalam format file excel. Pengguna juga dapat mengunduh file .pdf dengan klik cetak pada bagian atas tabel
  </Step>
</Steps>
