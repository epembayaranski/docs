---
title: "Registrasi"
description: "Rekap data pembayaran registrasi mahasiswa"
---

Menu laporan pembayaran registrasi digunakan admin untuk mendapatkan informasi seluruh aktivitas pembayaran registrasi mahasiswa. Admin dapat mengikuti langkah-langkah berikut untuk mendapatkan data laporan pembayaran registrasi.

<Steps>
  <Step title="Masuk ke aplikasi ePembayaran">
    
  </Step>
  <Step title="Pilih menu laporan pembayaran, pilih registrasi">
    ![Halaman Laporan Registrasi Pn](/images/halaman_laporan_registrasi.png)

    Gambar. Halaman Laporan Pembayaran Registrasi
  </Step>
  <Step title="Pilih filter data yang tersedia untuk mendapatkan data yang dibutuhkan">
    Filter yang tersedia pada menu laporan pembayaran registrasi adalah sebagai berikut

    - Tahun ajaran masuk
    - Jenjang Akademuk
    - Jalur masuk
    - Jenis Pembayaran
    - Tanggal awal pembayaran
    - Tanggal akhir pembayaran

    ![Filter Laporan Pembayaran Registrasi Pn](/images/filter_laporan_pembayaran_registrasi.png)

    Gambar. filter laporan pembayaran registrasi
  </Step>
  <Step title="Admin juga dapat melakukan export data laporan pembayaran registrasi">
    Setelah memilih filter data yang dibutuhkan admin dapat melakukan export data dengan klik cetak laporan, file dengan format .pdf akan otomatis ter-unduh berisi data sesuai yang dipilih sebelumnya.
  </Step>
</Steps>
