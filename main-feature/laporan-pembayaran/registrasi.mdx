Menu laporan pembayaran registrasi digunakan admin untuk mendapatkan informasi seluruh aktivitas pembayaran registrasi mahasiswa. Pengguna akan mendapatkan rekap per program studi mengenai data pembayaran registrasi. Pengguna juga dapat mencetak laporan dalam format file .pdf.

### Ikuti langkah-langkah berikut untuk mendapatkan data laporan pembayaran registrasi.

<Steps>
  <Step title="Masuk ke aplikasi ePembayaran">
    
  </Step>
  <Step title="Pilih menu laporan pembayaran, pilih registrasi">
    ![Halaman Laporan Registrasi Pn](/images/halaman_laporan_registrasi.png)

    Gambar. Halaman Laporan Pembayaran Registrasi
  </Step>
  <Step title="Pilih filter data yang tersedia untuk mendapatkan data yang dibutuhkan">
    Filter yang tersedia pada menu laporan pembayaran registrasi adalah sebagai berikut

    - Tahun ajaran masuk
    - Jenjang Akademuk
    - Jalur masuk
    - Jenis Pembayaran
    - Tanggal awal pembayaran
    - Tanggal akhir pembayaran

    ![Filter Laporan Pembayaran Registrasi Pn](/images/filter_laporan_pembayaran_registrasi.png)

    Gambar. filter laporan pembayaran registrasi
  </Step>
  <Step title="Admin juga dapat melakukan export data laporan pembayaran registrasi">
    Setelah memilih filter data yang dibutuhkan, admin dapat melakukan ekspor data dengan mengklik Cetak Laporan. File dengan format .pdf akan otomatis terunduh berisi data sesuai yang dipilih sebelumnya.
  </Step>
</Steps>
