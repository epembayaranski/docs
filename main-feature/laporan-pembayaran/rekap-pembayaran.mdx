Fitur Laporan Pembayaran pada aplikasi ePembayaran digunakan untuk melihat rekapitulasi transaksi pembayaran mahasiswa secara terstruktur. Pengguna juga dapat melihat transaksi berdasarkan operator mana yang melakukan verifikasi pembayaran, termasuk bank mana jika auto verifikasi diatur menjadi on.

### Untuk mengakses fitur ini anda dapat mengikuti langkah berikut:

<Steps>
  <Step title="Login ke aplikasi ePembayaran. Setelah berhasil masuk, pilih menu Laporan Pembayaran pada sidebar">
    
  </Step>
  <Step title="Klik sub menu Rekap Pembayaran. Pada halaman ini, user dapat menentukan filter data sesuai kebutuhan, seperti periode, jenis pembayaran, atau kriteria lainnya. Setelah filter diterapkan, sistem akan menampilkan jumlah total pembayaran berdasarkan hasil filter yang dipilih. ">
    
  </Step>
  <Step title="Selanjutnya, pada tabel rekap pembayaran akan ditampilkan data transaksi yang mencakup nomor urut, NIM, nama mahasiswa, nomor kuitansi, periode, jenis pembayaran, jumlah pembayaran, serta operator. ">
    
  </Step>
  <Step title="  Apabila user mengklik tombol aksi pada salah satu baris data, sistem akan menampilkan detail pembayaran mahasiswa secara lengkap untuk membantu proses pengecekan dan validasi transaksi.">
    
  </Step>
  <Step title="User juga dapat mengunduh data rekap pembayaran dalam format Excel (.xls) dengan menekan tombol Export .xls, sehingga data dapat disimpan dan diolah lebih lanjut.">
    
  </Step>
</Steps>
