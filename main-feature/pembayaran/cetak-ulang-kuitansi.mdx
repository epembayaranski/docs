Fitur ini digunakan untuk melakukan export kuitansi dari pembayaran yang sudah dilakukan oleh mahasiswa. Kuintansi berupa file .pdf berisi detail pembayaran mahasiswa seperti nama, nim, nomor pembayaran, jumlah pembayaran, jenis pembayaran, status pembayaran dan keterangan waktu pembayaran.

Pengguna dapat melakukan cetak kuitansi dengan mengikuti langkah-langkah berikut:

  </Step>

  </Step>
  <Step title="Setelah muncul halaman cetak ulang kuitansi, pilih filter data untuk menampilkan data pembayaran sesuai yang dibutuhkan, klik tombol tampilkan untuk menampilkan data">

    Gambar. Pop up data detail kuitansi
