Pada fitur data token pendaftaran, pengguna dapat melihat daftar token yang telah di buat pada menu token pendaftaran dan melihat sisa token yang belum digunakan atau dibeli oleh pendaftar di portal admisi.

Pengguna dapat membuka menu ini dengan memilih menu pembayaran, pilih menu data token pendaftaran, lalu pengguna dapat melihat list token lengkap dengan detail seperti Jalur, Gelombang, Kelompok Ujian, Jenis Pembayaran, Periode, Masa Pendaftaran, Sisa Token.

Gambar. Halaman Data Token Pendaftaran

<Note>
  **Data token pendaftaran** dapat berfungsi sebagai menu untuk melakukan <u>pembelian token pendaftaran</u>, hal ini dimungkinkan apablia config alur pendaftaran yang diaktifkan adalah alur tanpa pembelian mandiri oleh pendaftar di eAdmisi. Apabila config alur pembelian token pendaftaran melalui backoffice diaktifkan, maka pembelian dilakukan dengan klik tombol beli token, lalu admin mengisi detail informasi pendaftar termasuk nama dan kontak, lalu akan ter-generate akun login eAdmisi pendaftar, sehingga admin dapat memberikan akun tersebut ke calon mahasiswa yang mendaftar.
</Note>
