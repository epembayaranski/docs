Fitur ini dapat digunakan oleh pengguna untuk menyeleksi data dan menghapus data tagihan yang telah dibuat sebelumnya. Pengguna dapat mengikuti langkah berikut ini untuk menghapus data tagihan secara kolektif (masal).

### Langkah-langkah untuk menghapus tagihan secara kolektif:

<Steps>
  <Step title="Masuk ke aplikasi ePembayaran">
    
  </Step>
  <Step title="Pilih menu tagihan, pilih hapus tagihan kolektif">
    ![Hapustagihankolektifpage Pn](/images/hapustagihankolektifpage.png)

    Gambar. Halaman Hapus Tagihan Kolektif
  </Step>
  <Step title="Filter data yang dibutuhkan, klik tampilkan">
    Filter yang disediakan untuk menyeleksi data yaitu:

    - Periode Tagihan
    - Angkatan Mahasiswa
    - Jenis Pembayaran
    - Jenjang
    - Fakultas
    - Program Studi
    - Gelombang
    - UKT
    - Jalur Masuk
    - Status Mahasiswa
    - Waktu Kuliah
    - Status Edit

    ![Filter Hapus Tagihan Kolektif Pn](/images/filter_hapus_tagihan_kolektif.png)

    Gambar. Filter Hapus Tagihan Kolektif
  </Step>
  <Step title="Pilih tagihan yang akan dihapus dengan klik checkbox pada setiap daftar tagihan, pengguna juga bisa memilih semua dengan klik pilih semua (checkbox), klik tombol hapus untuk menghapus data">
    ![Hapustagihankolektifcheckboxselected](/images/Hapustagihankolektifcheckboxselected.png)

    Gambar. Daftar detail tagihan pembayaran mahasisw
  </Step>
  <Step title="Setelah klik tombol hapus, akan muncul pop-up konfirmasi, klik hapus untuk melajutkan proses hapus tagihan">
    ![Hapustagihankolektifcheckboxselected 1](/images/Hapustagihankolektifcheckboxselected-1.png)

    Gambar. Pop up konfirmasi hapus tagihan kolektif
  </Step>
</Steps>
