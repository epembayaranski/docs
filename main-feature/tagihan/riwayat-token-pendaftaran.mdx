---
title: "Riwayat Token Pendaftaran"
description: "Daftar data riwayat token pembayaran yang telah dibuat"
---

Pada fitur data token pembayaran, pengguna dapat melihat daftar token yang telah di buat pada menu token pendaftaran

### Langkah-langkah untuk melakukan unverifikasi pembayaran adalah sebagai berikut.

<Steps>
  <Step title="Masuk ke aplikasi ePembayaran">
    Login menggunakan akun pengguna ePembayaran
  </Step>
  <Step title="Pilih menu tagihan, pilih riwayat token pendaftaran">
    ![Halaman Riwayat Token Pendaftaran](/images/HalamanRiwayatTokenPendaftaran.png)
  </Step>
  <Step title="Setelah masuk ke halaman riwayat token pendaftaran, pilih filter yang tesedia untuk menampilkan data sesuai yang dibutuhkan, klik tombol tampilkan">
    **Filter yang disediakan yaitu:**

    ![Filter Riwayat Token Pendaftaran](/images/FilterRiwayatTokenPendaftaran.png)

    Periode : Pilih periode pembayaran (semester)

    Gelombang : Pilih gelombang sesuai pendaftaran yang tersedia

    Jalur Masuk : Pilih jalur masuk sesuai pendaftaran yang tersedia

    Kelompok Ujian: Pilih Kelompok ujian sesuai pendaftaran yang tersedia

    Status Token: Pilih status token sesuai pendaftaran yang tersedia

    Tanggal Awal Generate Token: Pilih tanggal awal generate token

    Tanggal Akhir Generate Token: Pilih tanggal akhir generate token

    Tanggal Awal Daftar: Pilih tanggal awal generate token

    Tanggal Akhir Daftar: Pilih tanggal akhir generate token
  </Step>
  <Step title="Setelah data muncul, pengguna dapat melihat seluruh data token pendaftaran yang telah dibuat sesuai filter yang dipilih, dan juga daftar token yang sudah dibeli dan belum di beli">
    Terdapat 2 jenis token yang terdapat di riwayat token pendaftaran

    1. Daftar token terbeli

       Daftar token terbeli akan berisi calon mahasiswa yang berhasil mendaftar melalui portal admisi di periode pendaftaran yang dibuat.

       Tampilan daftar token terbeli adalah sebagai berikut:

       ![Daftartokenterbelidiriwayattokenpendaftaran](/images/Daftartokenterbelidiriwayattokenpendaftaran.png)

       Pengguna juga dapat melakukan beberapa aksi pada daftar token terbeli, antara lain:
       - Edit token pendaftaran (edit detail pendaftar) dengan klik tombol edit pada kolom aksi
       - Print kuitansi, klik tombol print kuitansi pada kolom aksi
       - Print Bukti Booking, klik tombol Bukti Booking pada kolom aksi
    2. Daftar token belum terbeli

       Daftar token belum terbeli adalah token yang sudah di buat pada menu token pendaftaran, sesuai dengan jumlah kuota yang dibuat, namun token pendaftaran tersebut belum dibeli oleh calon mahasiswa melalui portal admisi.

       Tampilan daftar token terbeli adalah sebagai berikut:

       ![Tokenbelumterbelidiriwayattokenpendaftaran](/images/tokenbelumterbelidiriwayattokenpendaftaran.png)

    Pengguna juga dapat melakukan export data dengan file ekstensi excel (.xls), setelah memilih data dengan filter yang tersedia, pengguna klik tombol \*\*export .xls \*\*maka otomatis file excel akan terunduh di penyimpanan komputer.
  </Step>
</Steps>
