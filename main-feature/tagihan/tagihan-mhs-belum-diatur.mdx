Pengguna dapat melakukan rekap data program studi yang belum mengatur tagihan pada jenis pembayaran tertentu dan periode pembayaran tertentu.

Berikut langkah-langkah untuk melihat data program studi yang belum melakukan atur tagihan pembayaran mahasiswa:

<Steps>
  <Step title="Masuk ke aplikasi ePembayaran">
    
  </Step>
  <Step title="Pilih menu tagihan, pilih tagihan mahasiswa belum diatur">
    ![Halaman Tagihan Mahasiswa Belum Diatur](/images/HalamanTagihanMahasiswaBelumDiatur.png)

    Gambar. Halaman Tagihan Mahasiswa Belum Diatur
  </Step>
  <Step title="Pilih filter yang disediakan untuk menampilkan data sesuai kebutuhan, klik tombol tampilkan untuk menampilkan data">
    ![Filtertagihanmahasiswabelumdiatur](/images/filtertagihanmahasiswabelumdiatur.png)

    Gambar. Filter tagihan mahasiswa belum diatur
  </Step>
  <Step title="Data menampilkan jumlah data pembayaran (mahasiswa), pengguna dapat melihat detail tagihan dan melakukan export data dengan ekstensi file .xls">
    Klik tombol detail pada kolom aksi untuk melihat detail tagihan berisi list mahasiswa yang belum diatur tagihannya 

    ![Detailtagihanmahasiswabelumdiatur 1](/images/Detailtagihanmahasiswabelumdiatur-1.png)

    Gambar. Detail daftar tagihan belum diatur

    ![Downloadtagihanmahasiswabelumdiatur](/images/downloadtagihanmahasiswabelumdiatur.png)

    Gambar. Tombol download untuk mengunduh excel file data tagihan mahasiswa belum diatur
  </Step>
</Steps>
